# Hotel Management System (C++)

## Overview

Hotel Management System is a console-based application developed using C++. The system allows users to manage hotel room reservations, check-out processes, booking history, and receipt generation. Data is stored using text files to ensure information can be saved and retrieved between program executions.

## Features

* Display available hotel rooms
* Book a room for customers
* Check out occupied rooms
* Generate booking receipts
* Store booking history
* Save room status using file handling
* Simple menu-driven interface

## Technologies Used

* C++
* Structures
* Arrays
* File Handling (`fstream`)
* Object-Oriented Programming Concepts
* Text File Storage

## How to Run

1. Open the project in a C++ IDE (Code::Blocks, Dev-C++, Visual Studio, etc.).
2. Compile the source code.
3. Run the executable file.
4. Ensure that `rooms.txt` is located in the same directory as the program.
